# 清華大學校友高爾夫球隊賽事管理系統

## 系統概述
這是一個專為清華大學校友高爾夫球隊開發的賽事管理系統。系統採用現代化的網頁應用架構，提供直觀的使用者介面，協助管理球隊賽事、會員資料、成績記錄等各項功能。

## 系統版本
當前版本：V4.0
更新日期：2024-02-14

## 技術架構
### 前端技術
- React.js
- Material-UI (MUI)
- Axios HTTP 客戶端
- 響應式設計，支援多種設備

### 後端技術
- Python Flask 框架
- PostgreSQL 資料庫
- SQLAlchemy ORM
- Flask-Migrate 資料庫遷移
- RESTful API 設計

### 部署環境
- 前端：Render.com 靜態網站託管
- 後端：Render.com Web Service
- 資料庫：Render.com PostgreSQL

## 系統功能

### A. 會員管理
1. 會員資料管理
   - 支援 Excel 批量上傳會員資料
   - 會員資料版本控制
   - 版本差異比對
   - 單筆會員新增/編輯/刪除
   - 批量刪除功能
   - 會員資料匯出

2. 會員資料欄位
   - 會員編號
   - 中文姓名
   - 英文姓名
   - 系級
   - 會員類型（會員/來賓）
   - 性別
   - 差點
   - 管理員權限

### B. 賽事管理
1. 賽事基本資料
   - 賽事名稱
   - 日期
   - 場地
   - 備註

2. 賽事獎項管理
   - 技術獎（一般組/長青組）
   - 跳跳獎
   - 總桿冠軍
   - 淨桿獎（1-10名）
   - 會長獎
   - BB獎
   - Eagle獎
   - HIO
   - 其他客製化獎項

### C. 成績管理
1. 成績記錄功能
   - Excel 成績單批量上傳
   - 自動計算差點
   - 自動計算積分
   - 成績資料匯出

2. 成績查詢
   - 依賽事查詢
   - 依會員查詢
   - 年度統計
   - 積分排名

### D. 報表分析
1. 統計報表
   - 各組平均差點統計
   - 各組平均桿數統計
   - 積分排名前 10 名
   - 總桿數前 10 名
   - 進步最多前 5 名
   - 全勤獎統計

2. 資料匯出
   - Excel 格式匯出
   - 自訂報表範圍

### E. 系統管理
1. 儀表板
   - 系統概況
   - 最新公告
   - 近期賽事
   - 年度冠軍榜

2. 版本管理
   - 版本更新記錄
   - 系統說明

### F. 遊戲功能
- 轉盤抽獎遊戲
- 可自訂獎項
- 動態效果

## 安裝說明

### 後端設置
1. 安裝 Python 依賴套件：
```bash
pip install -r requirements.txt
```

2. 設置環境變數：
```bash
cp .env.example .env
# 編輯 .env 文件，設置必要的環境變數
```

3. 初始化資料庫：
```bash
python init_db_with_migrate.py
```

4. 初始化獎項類型：
```bash
python init_award_types.py
```

5. 運行開發伺服器：
```bash
flask run
```

### 前端設置
1. 安裝 Node.js 依賴：
```bash
cd frontend
npm install
```

2. 設置環境變數：
```bash
cp .env.example .env
# 編輯 .env 文件，設置 API 位址
```

3. 運行開發伺服器：
```bash
npm start
```

## 部署說明
1. 後端部署
   - 使用 Render.com Web Service
   - 設置環境變數
   - 設置啟動命令：`gunicorn "app:create_app()" --timeout 120`

2. 前端部署
   - 使用 Render.com 靜態網站託管
   - 設置建構命令：`npm run build`
   - 設置發布目錄：`build`
